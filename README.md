# QR Code Web Shaper

A simple Next.js web application for generating QR codes from links or text.

## Features

- Enter any URL or text
- Real-time QR code preview
- Download QR code as PNG image

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- qrcode library
